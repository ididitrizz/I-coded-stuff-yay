This is a blackjack game coded in the Python engine.
It should work in the 3.14.2 version.
If you don't have a Python app go here and upload the blackjack.py file: https://www.online-python.com/

RULES: \n
Attempt to get as high of a hand value as possible without going over 21 \n
J,Q and K are worth 10 \n
Number cards are worth their value \n
A is worth either 11 or 1, the game will adjust this automatically. \n
Getting 21 with two cards is worth more than getting 21 with more cards. \n
Try to beat the dealer; dealer wins on a tie. \n

That's pretty much it, have fun!
